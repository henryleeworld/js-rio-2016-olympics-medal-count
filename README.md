# JS 2016 里約奧運獎牌統計

2016 年奧運共有 28 種運動大項，包含 306 樣競賽項目，其中 7 人制橄欖球取代 15 人制橄欖球首度成為奧運項目，而高爾夫球運動則是暌違 112 年後重返最高殿堂。

## 使用方式
- 把整個專案複製一份到你的電腦裡，這裡指的「內容」不是只有檔案，而是指所有整個專案的歷史紀錄、分支、標籤等內容都會複製一份下來。
```sh
$ git clone
```
